## In this project, the following technologies were utilized:

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** Node.js
- **API:** https://www.themoviedb.org/
- **Version Control:** Git, GitHub
- **Bundler:** Vite
- **Design Tools:** Figma
